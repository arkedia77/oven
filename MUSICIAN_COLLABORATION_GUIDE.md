## 1️⃣ 협업 프로세스

### Step 1: 대상자 선정 & 제안
```
1. 현장 만남 또는 온라인 제안
2. 프로젝트 설명
   - "AI 음악 엔진" 비전 공유
   - 당신의 역할 & 데이터 활용
3. PIANIST_INTERVIEW.md 같은 구체적 가이드 제시
4. 협업 관심도 확인
```

### Step 2: 인터뷰 (1시간 ~ 2시간)
```
1. 악기별 표현 기법 상세 인터뷰
2. 핵심 질문 5개 (시간 없으면 이것만!)
3. 메모 + 음성 녹음 (허락 하에)
4. 선곡 리스트 확보
```

### Step 3: MIDI 레코딩 (선택사항, 최고 가치)
```
준비물:
- 디지털 피아노 또는 악기 + MIDI 인터페이스
- DAW (Logic, Ableton, Cubase 등)
- 안정적 인터넷 (온라인 협업 시)

절차:
1. 같은 곡 3~5가지 스타일로 연주
   예: "몽드리안의 꿈"
   - 클래식 스타일 (보수적)
   - 팝 스타일 (자유로움)
   - 당신 스타일 (개인 표현)

2. MIDI 파일 수집
   - 악기별 MIDI 채널 분리
   - 페달/컨트롤러 신호 포함
   - 여러 템포 버전 (느림/중간/빠름)

3. 메타데이터 수집
   - 곡명, 작곡가, 난이도
   - 각 버전의 특징 (스타일, 템포, 분위기)
   - 연주자 노트 (특별한 기법)
```

### Step 4: 데이터 처리 & 분석
```
1. MIDI 데이터 정규화
   - 온타임 분석 (루바토)
   - 벨로시티 히스토그램
   - 페달 사용 패턴
   - 아티큘레이션 추출

2. 프로필 작성
   → MUSICIAN_PROFILE.md
   - 개인 스타일 요약
   - 표현 특징 (수치)
   - 추천 레퍼런스 곡

3. AI 파인튜닝 데이터 구성
   - 피아노 솔로 데이터셋 구축
   - 악기별 성능 메트릭
```

### Step 5: 피드백 & 협업 심화
```
1. AI 생성 결과물 공유
   - 당신의 스타일로 생성된 MIDI
   - 실제 연주와 비교

2. 피드백 수집
   - "어떤 부분이 너의 표현을 잘 따라 했나?"
   - 개선할 점은?
   - 흥미로운 발견?

3. 반복 파인튜닝
   - 피드백 반영
   - 다음 버전 생성

4. 장기 협업 논의
   - Phase 5 트로트 모델?
   - 퍼블릭 공개?
   - 수익 배분?
```

3. 포트폴리오 가치

## 3️⃣ 악기별 협업 템플릿

### 피아노 (완성됨)
- 가이드: `PIANIST_INTERVIEW.md`
- 데이터: MIDI 레코딩
- 학습 대상: 루바토, 벨로시티, 페달링

### 기타 (향후)
```markdown
# 기타리스트 인터뷰 아젠다
## 핵심 표현
- 비브라토 (깊이, 속도, 타이밍)
- 벤딩 (반음 vs 1음, 속도)
- 스트럼 패턴 (다운/업 비율)
- 필터 + 이펙트 (프로그레션)

## 데이터 수집
- MIDI 레코딩: 같은 곡 3가지 스타일
- 영상: 손/피킹 기법 분석
```

### 보컬 (향후)
```markdown
# 가수 협업 프레임워크
## 핵심 표현
- 음정 미세조정 (비브라토, 글리산도)
- 발음 + 감정
- 호흡 타이밍

## 데이터 수집
- 오디오 녹음 (48kHz 이상)
- Demucs로 보컬 분리
- Basic Pitch로 MIDI화
```
